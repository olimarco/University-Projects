# University Projects

Welcome to my repository. This space serves as a curated portfolio of my academic work as an Artificial Intelligence student. It is designed to showcase the practical application of theoretical concepts through complex, well-structured software development.

Rather than a simple storage folder, this repository is built using **Git Submodules**. This architecture preserves the individual commit history and standalone integrity of each exam project, while providing a unified entry point for review.

## Featured Projects

Below are the main projects currently linked in this repository:

*   **[NBA Players Stats Analysis](./NBA-Players-Stats-Analysis)**
    *   **Domain:** Data Science & Machine Learning.
    *   **Description:** An in-depth analysis of a basketball statistics dataset. The project focuses on feature selection, data visualization, and the mathematical implementation of both linear and non-linear regression models to uncover underlying data patterns.
    *   **Tech Stack:** Python, Jupyter Notebook, Data Science Libraries (Pandas, Matplotlib).

*   **[Galactic Colonization / Deep Space Explorer](./Galactic-Colonization)**
    *   **Domain:** Algorithms & Data Structures.
    *   **Description:** A complex space exploration simulator emphasizing resource management and procedural universe generation. It heavily utilizes graph data structures and advanced search algorithms (such as DFS with backtracking) to navigate uncharted galactic sectors efficiently.
    *   **Tech Stack:** Python, Object-Oriented Programming (OOP).

*   **[Hidden Treasures](./Hidden-Treasures)**
    *   **Domain:** Software Architecture & Game Agent.
    *   **Description:** A strategic, turn-based card game played on a 6x6 grid, emphasizing resource management (Action Points) and an adapted poker-style scoring system. Built upon a rigorous Model-View-Controller (MVC) architecture, the project features a custom graphical user interface and includes a single-player mode powered by a heuristic state machine bot capable of memory retention and dynamic strategy pivoting.
    *   **Tech Stack:** Python, MVC Pattern, GUI Programming (breezypythongui, Tkinter).
